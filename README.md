# @etrepum/lexical-builder

The end product of Lexical Builder is the new
[Lexical Extension](https://lexical.dev/docs/extensions/intro) feature
released in
[Lexical v0.36.1](https://github.com/facebook/lexical/releases/tag/v0.36.1).
This project will be archived when the remainder of the example code is
ported.

https://lexical-builder.pages.dev/

[EXPERIMENTAL] A high-level way to manage Lexical configuration and plug-ins
for any framework

WARNING: This is a prototype. Do not use it. The expectation is that it will
either adopted into the Lexical monorepo as a de jure standard, or abandoned.
